Ansible provides open-source automation that reduces complexity and allows automation for the cluster node setup.

::: info Why should we use Ansible?
* **Agent-less architecture:** No additional install mainten across IT infrastructure.

* **Simplicity:** Easy to read .yaml, decentralized using SSH to access nodes.

* **Scalability and flexibility:** Modular design and wide range of supported platforms.

:::
